## ğŸš€ Getting Started

### Prerequisites

- Node.js v18+ ([Download here](https://nodejs.org/))
- npm 9+ (comes with Node.js)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/CyberSecure-Toolkit.git
   cd CyberSecure-Toolkit/frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   - Navigate to `http://localhost:5173/`
